2D Zombie Survival Game
Title Description:
A simple 2D survival game where the player fights zombies, collects health packs, and survives as long as possible.
Tools / Resources:
•	Java / C++
•	Java Swing or SFML
•	Game Loop Logic
•	Keyboard Event Handling
Steps:
1.	Player movement system
2.	Enemy zombie spawning
3.	Collision detection
4.	Health and score system
5.	Game over system
Architecture:
Player Input → Game Engine → Collision Detection → Score System → Display
Outcome:
A fun and interactive survival game that improves game development and logic-building skills.
